Pour le salarié·e qui utilise cet outil
Cet outil vous permet de vérifier si votre rémunération respecte les minima de votre convention collective. Si une alerte apparaît, cela signifie qu'un écart de plus de 5 % est constaté par rapport à la moyenne du groupe de postes comparables. Ce n'est pas forcément une erreur — cela invite à un échange avec votre employeur pour clarifier les critères de votre positionnement (responsabilités, compétences, ancienneté). Toutes vos données restent dans votre navigateur, rien n'est transmis.
