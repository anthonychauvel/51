📌 Règles essentielles

  • Chômés et payés (Art. L3133-1) : les 11 jours fériés légaux sont obligatoirement chômés pour les salariés dont l'horaire habituel comprend ce jour.
  • Pont non obligatoire : le pont (ex. vendredi entre un jeudi férié et le week-end) n'est pas de droit. Il doit être négocié ou compensé.
  • Majoration 100 % : si le salarié travaille un jour férié (hors 1er mai), la convention collective ou l'accord d'entreprise peut prévoir une majoration. Le 1er mai travaillé est obligatoirement majoré de 100 %.
  • Jours fériés tombant le dimanche : pas de report obligatoire sauf dispositions conventionnelles.
  • Alsace-Moselle : 2 jours fériés supplémentaires (Vendredi Saint + 26 décembre — St-Étienne).

⚖️ Droit au chômage du jour férié

Pour bénéficier du chômage d'un jour férié sans perte de salaire, le salarié doit justifier de 3 mois d'ancienneté dans l'entreprise (ou d'accord collectif plus favorable). Les intérimaires bénéficient des mêmes droits que les salariés permanents pour les jours fériés travaillés.
